## 🔄 Fluxo da Pipeline

`attgeral()` executa:

1. Importação de agenda
2. Atualização de referências
3. Atualização de etapas e consultoras
4. Criação de abas por ferramenta
5. Consolidação final para BI

---

## 📈 Módulos suportados

- Analytics
- Força de Vendas
- CRM
- Portal B2B

Cada módulo calcula:

- Engajamento
- Parcialmente Engajado
- Não Engajado
- Sem Ferramenta

Com base em pesos configuráveis.
